    Writing Tutorial

    Where I exhibit simple basic writing features that Miniedit has.

    These include:

  • Arrow keys to navigate your cursor around text

  • ctrl+arrow keys to jump words

  • regular keys

    • Oh:

  • Ctrl+backspace is for removing a word.

    • Some Cool Commands:

    Commands are sheltered with []

    Your First Commands:

  • Equals command to replace a word with another: [first words equals second words 123?] //you need to place this atop the file, others not so much

  • [caps some words here] command to capitalize words. You could wrap paragraphs with it.

  • Date command to write a date: [date]

  • Swap command to swap some text every few seconds [firstword swap secondword]

  • [utf65] to have a utf8 character. Which means you could write β with [utf946] (may refuse to display though.)

  • [emdash] to create — (it's a utf8 character)

  • [alpha] to create α (it's a utf8 character)

  • Look out for Cool Features to see more utf8 shortcuts.

    • Font Stuff:

  • [regular] to have regular font

  • [bold] to have bold font

  • [italic] to have italic font

  • [monospace] to have monospace font
