    Writing Tutorial

    Where I exhibit simple basic writing features that Miniedit has.

    These include:

  • Arrow keys to navigate your cursor around text

  • ctrl+arrow keys to jump words

  • regular keys

    • Oh:

  • Ctrl+backspace is for removing a word.

    • Some Cool Commands:

    Commands are sheltered with []

  • Equals command to replace a word with another: [firstword equals secondword] //you need to place this atop the file, others not so much

  • Date command to write a date: [date]

  • Swap command to swap some text every few seconds [firstword swap secondword]
