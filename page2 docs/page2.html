    UI Tutorial

    Introduction

    Where I explain miniedit's structure

    When you open miniedit,

    you meet a mini window.

    At the top,

    you have the menu.

    FILE > new creates a new file.

    Type the file name in the "Edit file name" textbox.

    You save the file with FILE > save.

    You open the file with FILE > open.

    ALL FILES WORK WITH THE SAME DIRECTORY AS THE EXE FILE.

    EDIT > Preferences has Full Screen toggle, and more.

    Move mouse to right to reveal the writing area (text area).

    Hot key for exiting the program is "ESC"
